模块名称： Fixed

使元素处于一个绝对位置，不受页面滚动影响。

使用方法

使用此功能模块，需要在页面下面引入如下代码： 

<script type="text/javascript"  src="../images/jquery-1.8.3.min.js" charset="UTF-8"> </script>
<script type="text/javascript" src="../images/qdframe-v1.0.0.js" charset="UTF-8"></script>

引入说明

代码执行方法 
<script>
   QD.fix (id , type , className); 
   QD.fixRemove (id , type , className); //type 参数暂不支持像素值
</script>

参数说明


Id : 需要固定元素ID (必填)

type: 元素固定位置( top ,bottom,left , right , center ,middle ,或像素值)2个位置用逗号隔开,当位置为像素值时遵循top,letf的顺序（必填 多选）

className : 还需要往元素上添加的class多个class用空格隔开（选填）

例子代码

 // 
<script>
QD.fix("right","right,bottom"); //元素固定在右下角
QD.fix("top","top");//元素固定在左上角
QD.fix("bottom","bottom");//左下角
QD.fix("center","top,center");//底部水平居中
QD.fixRemove("center");//移除元素的所有位置
QD.fix("middle","center,middle","green");//在页面中央 
QD.fix("abs","100,500");//固定位置
</script>
