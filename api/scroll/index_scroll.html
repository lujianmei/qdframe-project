模块名称： Scroll

功能描述：鼠标指向，根据位置左右移动；(上下形式将上下移动) 点击按钮左右移动；(点击上下按钮上下移动)

使用方法

使用此功能模块，需要在页面下面引入如下代码： 

<script type="text/javascript"  src="../images/jquery-1.8.3.min.js" charset="UTF-8"> </script>
<script type="text/javascript" src="../images/qdframe-v1.0.0.js" charset="UTF-8"></script>

引入说明

代码执行方法 
<script>
QD.Scroll(idobj,scrollnum,timer,hidebtn,prevclassname,nextclassname,scrolldirect,istoscroll); 
</script>

参数说明

idobj:表示要滚动的容器对象

scrollnum:表示一次滚动的数目，参考值为1

timer:表示鼠标以上多长时间滚动到尽头，单位为毫秒，参考值为4000

hidebtn:表示是否隐藏左右/上下滚动按钮0表示不隐藏，1表示隐藏，参考值为 1

prevclassname:表示左/上按钮的类名，便于前端开发人员自由传递定义的自己的类名，参考值为js_scroll_prev

nextclassname:表示右/下按钮的类名，便于前端开发人员自由传递定义的自己的类名，参考值为js_scroll_next

scrolldirect:表示滚动方向，0表示左右滚动 1表示上下滚动 当然左右滚动还是上下滚动前端通过样式控制，参考值为0

istoscroll:表示鼠标移向是否滚动，1表示滚动 0表示不滚动   参考值为 1

例子代码

//左右滚动
QD.Scroll($(".js_scroll_products"),1,3000,0,"js_scroll_prev","js_scroll_next",0,1); 
//上下滚动
QD.Scroll($(".js_scroll_products"),1,3000,0,"js_scroll_prev","js_scroll_next",1,0);
