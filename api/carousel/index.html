模块名称： Carousel

功能描述
页面轮播图片

使用方法

使用此功能模块，需要在页面下面引入如下代码： 

<script type="text/javascript"  src="../images/jquery-1.8.3.min.js" charset="UTF-8"> </script>
<script type="text/javascript" src="../images/qdframe-v1.0.0.js" charset="UTF-8"></script>

引入说明

代码执行方法 
<script>
	QD.Carousel(2000,true,112);//注册主视觉事件
</script>

参数说明


	pram1:	//播放时间
	pram2:	//是否自动播放
	pram3:	//轮播按钮宽度 值等于0是不设置宽度
	pram4:	//轮播图片选择器
	pram5:	//轮播按钮选择器

例子代码

 // 例子代码
说明
	QD.Carousel(2000,true,112,$(".js_kv_item li"),$(".js_kv_nav li"));//注册主视觉事件
