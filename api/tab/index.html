模块名称： Tab changing

功能描述 用于网页建设tab切换内容使用，功能分两种使用方式，异步请求数据和非异步请求数据，异步请求需要使用Tomcat或其它服务器查看。

使用方法

使用此功能模块，需要在页面下面引入如下代码： 

<script type="text/javascript"  src="../images/jquery-1.8.3.min.js" charset="UTF-8"> </script>
<script type="text/javascript" src="../images/qdframe-v1.0.0.js" charset="UTF-8"></script>

引入说明

代码执行方法 
<script>

QD.ChangeTab($(".js_sys_nav li a"),$(".js_drive_data_wrap .drive_data_item"),"active",false);

</script>

参数说明

pram1: //定义标题切换选择器
pram2: //定义内容切换选择器
pram3: //切换定义class
pram4: //是否异步请求

例子代码

 // 例子代码
说明
	QD.ChangeTab($(".js_sys_nav li a"),$(".js_drive_data_wrap .drive_data_item"),"active",false);

	异步请求HTML写法，增加rel属性，异步数据URL地址：

<ul class="sys_nav js_sys_navs">
	<li><a href="javascript:;" rel="../data/winxp.xml"><b>Windows XP</b></a></li>
	<li><a href="javascript:;" rel="../data/winvista.xml"><b>Windows Vista</b></a></li>
	<li><a href="javascript:;" rel="../data/win7.xml"><b>Windows 7</b></a></li>
	<li><a href="javascript:;" rel="../data/win8.xml"><b>Windows 8</b></a></li>
</ul>
	QD.ChangeTab($(".js_sys_nav li a"),$(".js_drive_data_wrap .drive_data_item"),"active",true);
